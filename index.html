The Basic Language of the Web:HTML Top

posted by laura jones on monday, june 21st 2027

all modern website and web application are built using three foundamental technologies HTML, CSS, Javascript.these are the language of the web.

In this post, let focus on HTML, we will learn what HTML is all about and why you too should learn it

what is HTML?

HTML stands for Hyper Text Markup language. its a markup language that web developers use to structure and describe the content of a webpage (not a programing language).

HTML consist of element that describe different types of content: paragraphs, links, heading, imges, videos, etc. web browser understand HTML and render HTML code as website.

in HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element
you can learn more at MDN Web Docs.
